# Grimgur
An image board with user submissions. Implementing Flexbox for responsiveness, and Media Queries for mobile/tablet devices.


## Technologies
- **Language**: Javascript
- **Frontend**: Angular, Handlebars
- **Backend**: Node, Express
- **Databases**: PostgreSQL, AWS S3
